# Feedback Alignment on XOR

This repository contains a simple implementation of **Feedback Alignment (FA)** on the **XOR problem**, along with supporting files for analysis, experiments, and visualization.

## Overview

The main script in this project is:

- `XOR.py` — runs the Feedback Alignment experiment on XOR

The repository also includes folders for analysis, miscellaneous files, sample measurement data, and OECT measurement scripts.

---

## Requirements

This project uses:

- **Python 3.10**

Install the required packages from `requirements.txt`.

---
